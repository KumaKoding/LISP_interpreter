# Basic LISP interpreter implemented in C

## Current Features
Currently, the only features are:
1. Input handling
2. Lexing / Tokenization
3. Parsing
4. Primitive Runtime

## Future Features
In the future, there will be:
3. Garbage Collection
4. Fully functioning scheme

## Instructions
1. Clone directory
2. run `make` in terminal
3. run `make run`
